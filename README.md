Proyek analisis dan klasifikasi kanker payudara menggunakan Machine Learning. 
Dataset diambil dari kaggle (https://www.kaggle.com/datasets/wasiqaliyasir/breast-cancer-dataset).
Pemodelan menggunakan algortima Decision Tree untuk mengidentifikasi apakah subjek positif kanker payudara atau tidak.
Visualisasi menggunakan matplotlib dan seaborn.

ROADMAP:
- Random Forest
- Perbandingan beberapa algoritma
- ROC Curve & F1-score
- Hyperparameter tuning
